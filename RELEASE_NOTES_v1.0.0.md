# The Parasites CorpseReaper v1.0.0

Initial public portable release.

## Download

Use the attached release ZIP:

```text
TheParasites_CorpseReaper_v1.0.0.zip
```

## Usage

1. Close The Parasites completely.
2. Back up your saves.
3. Extract the ZIP.
4. Run `Start_CorpseReaper_Prune_Prepare.cmd`.
5. Choose a slot, for example `savegame_5` or later `savegame_1`.
6. Load the prepared slot in-game.
7. Check that buildings, crafting stations, traps, and containers are still
   present.
8. Save normally once in-game.
9. Exit the game.
10. Optional analysis:

```text
Start_CorpseReaper.cmd --analyze savegame_1
```

## Purpose

Prepares compatible zombie DeathPose actor records for an in-game prune pass so
the game can stop writing those invalid zombie actors back into the save.

## Target Version

Prepared for:

```text
The Parasites TP_Alpha_v_0.1.5.0.0
```

No compatibility is promised for other game versions.
